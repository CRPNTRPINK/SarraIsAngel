# для Сарры ♡

Сайт-сюрприз: ввод пароля → анимация с облачками и кнопка «пуск» с записью.

## Как выложить на GitHub Pages

1. Откройте `encrypt.html` локально в браузере (двойной клик по файлу).
2. Загрузите туда свой `song.ogg`, введите пароль
3. Скачается `song.enc` — положите его в `assets/song.enc`.
4. **Не коммитьте** `assets/song.ogg` — он уже в `.gitignore`.
5. Коммит, push, включите GitHub Pages в настройках репозитория.

## Что в репозитории

```
index.html         — сайт
encrypt.html       — локальный шифровальщик (можно оставить, он безопасен)
assets/
  song.enc         — ← зашифрованная запись (кладёте сюда)
  cloud.png
  girl.png
.gitignore         — прячет оригинал song.ogg
```

## Безопасность

Аудио зашифровано **AES-256-GCM**, ключ выводится из пароля через PBKDF2 (200 000 итераций, SHA-256). Без правильного пароля расшифровать файл нельзя даже при полном доступе к репозиторию.

Если когда-нибудь измените пароль — перешифруйте файл через `encrypt.html` заново.
